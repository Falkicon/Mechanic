Research WoW APIs and patterns for addon development.

**Skill**: [s-research](../skills/s-research/SKILL.md)

1. **Search**: Use `api.search` MCP tool to find relevant WoW APIs by name or pattern.
2. **Inspect**: Use `api.info` MCP tool to get detailed documentation for specific APIs.
3. **Explore**: Browse `api.list` by namespace to understand related functions.
4. **Pattern**: Search Blizzard UI source or Ace3 patterns for usage examples.
5. **Document**: Summarize findings with code examples for the user.
